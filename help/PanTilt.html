PanTilt

Base class:

Generate servo commands for a pan/tilt head to track a target (e.g. for camera or antenna trackers)

Usage Guide

Example Configuration

PanTilt.new 21
  name "PanTilt"
  PID (f) 0.005 0.0 0.0001
  interval 50
  $target [@>20.8]
  $heading [@>6.11]
  .publish "target"
  .publish "heading"
  .publish "PID"
  .publish "pan"
.done

Subs (inputs)

Address Addr Address Name Type # Description
10 11 target f 1 Target heading (e.g. from Nav module)
PanTilt Generate servo commands for a pan/tilt head to track a target (e.g. for camera or antenna trackers)

Pubs (outputs / config)

Address Name Type # Description
14 PID f 3 PID values (start with: 0.005 0.0 0.0001)
15 limits f 2 Limits of servo travel in degrees (default -90 90)
16 pan f 1 Pan output in range -1..1, to be fed to servo
