Waypoint

Base class: Drone

Load and manage a series of waypoints from a CSV file

Usage Guide

Load and manage stepping through a series of waypoints, defined in /waypoint.csv.

Example:

lon,lat,radius
-1.744739150337,51.541523406305,10.0
-1.744404186599,51.542305699213,15.0
-1.746584868898,51.542284441432,10.0
-1.746516508951,51.541510651418,10.0

Example Configuration

[Waypoint=4]
  name=Waypoint
  $location=@>50.9
  publish =mode, waypoint, waypoints, target, location

Subs (inputs)

Address Addr Address Name Type # Description
12 13 location f 3 Location from GPS

Pubs (outputs / config)

Address Name Type # Description
8 mode u8 1 Mode 0=normal, 1=Reload waypoint.csv
9 waypoints u8 1 How many waypoints are loaded
10 waypoint u8 1 Current waypoint number
11 target f 3 Target location of current waypoint, feed to Nav.target
12 loop u8 1 0=stop at end of file, 1=loop
