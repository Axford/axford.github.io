DroneLink Firmware Setup Guide

  1. Compile and install latest firmware onto ESP32 using Platform.io over USB connection
  2. Open serial terminal to ESP at 115200 baud
  3. Wait until IP address is shown (obtained via DHCP) - indicated by the blue LED going out.
  4. Connect in web-browser to ESP IP address
  5. Click List Files top right - by default a startup.log and boot.dat files should have automatically been created. If nothing is displayed, then reboot the ESP and check again.
  6. Click Upload and upload config.txt and any other configuration files (e.g. main.txt, arm.txt, disarm.txt). If using a compass module, then upload an appropriate magnetic declination file (mag.dat).
  7. Ensure the new device is configured with a unique Node address and hostname
  8. Reboot the device using the Reboot button and wait for WIFI connection (blue LED goes out)
  9. Switch to DroneLink web interface and the new device should appear

Further configuration can be managed via the DroneLink web interface.

Troubleshooting

If filesystem issues occur, the LITTLEFS flash filesystem can be formatted from the ESP serial interface:

  1. Connect to the ESP serial interface from a terminal
  2. Type "format" and press Enter
  3. The internal filesystem will be erased and formatted.
  4. Reboot the device and re-configure the device by uploading configuration files.
