CMPS12

Base class: I2CBase

Manages a CMPS12 I2C Compass

Example Configuration

[CMPS12= 8]
  name= "Compass"
  interval= 100
  bus= 3
  status= 1
  trim= 0
  location= -1.8, 52, 100
  $location= @>7.8
  publish = heading, trim

Subs (inputs)

Address Addr Address Name Type # Description
20 21 location f 2 Current location from GPS

Pubs (outputs / config)

Address Name Type # Description
10 heading f 1 Heading adjusted for magnetic declination
11 declination f 1 Current declination value
12 trim f 1 Manual calibration value to adjust heading to match hull (e.g. for a misaligned physical mount)
13 vector f 2 Pitch and roll vector
