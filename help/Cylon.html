Cylon

Base class: Drone

Manage a strip of NEOPixels (WS2812B) - originally created for the Kit speedboat

Subs (inputs)

Address Addr Address Name Type # Description
8 9 left f 1 Left motor speed
10 11 right f 1 Right motor speed

Pubs (outputs / config)

Address Name Type # Description
12 pins u8 1 Which output pin is connected to the pixel strip
13 numPixels u8 1 How many pixels in the strip
