NMEA

Base class: I2CBase

Manages a serial NMEA GPS device

Example Configuration

[NMEA= 7]
  name= "GPS"
  interval= 500
  port= 2
  baud= 9600
  publish =location, satellites, HDOP, followMe

Subs (inputs)

Address Addr Address Name Type # Description
20 21 correction f 3 If fix location is set then this is the difference between live GPS and known location, for other nodes to subscribe to. If no fix location, then this can be subbed to a base station correction param.

Pubs (outputs / config)

Address Name Type # Description
8 location f 2 Current GPS location
9 satellites f 1 Number of satellites used for fix
10 heading f 1 Current heading over ground
11 speed f 1 Current speed over ground
12 HDOP f 1 Current Horizontal Dilution of Precision
13 port u8 1 Which serial port to use (0,1 or 2)
14 baud u32 1 Baud rate to use, normally 9600
15 fix f 3 Fix location - to use as basis for differential GPS signal
16 followMe f 3 Location field with a small target radius
17 packets u32 3 Number of NMEA packets (sentences) received (valid, invalid, unknown)
