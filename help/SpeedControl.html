SpeedControl

Base class: Drone

Convert a distance value (meters) into a speed control value (0..1)

Usage Guide

  • Sub to distance (from Nav)
  • Param for speed limits
  • Param for distance threshold at which to start slowing down
  • Generate speed output to feed TankSteer, etc

    Subs (inputs)

    Address Addr Address Name Type # Description
    10 11 distance f 1 Distance to target, typically Nav.distance

    Pubs (outputs / config)

    Address Name Type # Description
    12 limits f 2 Min and max output speed range
    13 threshold f 1 Distance in meters within which to throttle down to min limit
    14 speed f 1 Output speed
