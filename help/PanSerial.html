PanSerial

Base class:

Generate serial commands for a slave pan controller module

Usage Guide

Example Configuration

PanSerial.new 21
  name "PanSerial"
  PID (f) 0.005 0.0 0.0001
  interval 50
  $target [@>20.8]
  $heading [@>6.11]
  .publish "target"
  .publish "heading"
  .publish "PID"
  .publish "pan"
.done

Subs (inputs)

Address Addr Address Name Type # Description
10 11 target f 1 Target heading (e.g. from Nav module)
PanSerial Generate serial commands for a slave pan controller module

Pubs (outputs / config)

Address Name Type # Description
14 PID f 3 PID values (start with: 0.005 0.0 0.0001)
15 pan u8 1 Pan output in range 0..255 as sent over serial
