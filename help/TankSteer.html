TankSteer

Base class:

Generate left and right motor speeds from speed and turnRate inputs

Usage Guide

Tank steering: * Sub to target heading * Sub to current heading * Sub to distance * Generate left and right motor speed values

Example Configuration

TankSteer.new 10
  name "TankSteer"
  interval 50
  trim 0
  $speed [@>9.14]
  $turnRate [@>8.16]
  .publish "left"
  .publish "right"
  .publish "turnRate"
  .publish "speed"
  .publish "trim"
.done

Subs (inputs)

Address Addr Address Name Type # Description
10 11 turnRate f 1 Rate of turn in range -1..1 where positive values are clockwise
12 13 speed f 1 Desired speed in range -1 to 1
14 15 trim f 1 Applied to turnRate value

Pubs (outputs / config)

Address Name Type # Description
8 left f 1 Left motor speed output in range -1..1
9 right f 1 Right motor speed output in range -1..1
16 mode u8 1 1=Automated (responds to speed and turnRate subs) or 0=Manual (ignores subs, but speed and turnRate values can be directly written to)
