Servo

Base class: Drone

Manages a PWM servo channel

Example Configuration

[Servo= 13]
  position= 0
  pins= 4
  status= 1
  name= "Rudder"
  map= 50, 80, 100, 130
  centre= 8
  $position= @>11.16 
  interval= 100
  publish= map, centre, position, output

Subs (inputs)

Address Addr Address Name Type # Description
8 9 position f 1 The input position value (-1..1)

Pubs (outputs / config)

Address Name Type # Description
10 pins u8 1 Servo output pin, see Pin Mappings
11 limits f 1 Movement rate limit in degrees/second
12 map f 4 Bezier curve control points to map input (-1..1) to output (0..180)
13 centre f 1 Centre adjustment, applied after map
14 output f 1 The raw angle sent to the Servo PWM controller (0..180)
