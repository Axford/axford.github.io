ODrive

Base class: Drone

Manages an ODrive module via serial

Example Configuration

ODrive.new 11
  name "ODrive"
  interval 100
  port 2
  limits -10 10
  invert 0
  $left [@>10.8]
  $right [@>10.9]

  // publish
  .publish "left"
  .publish "right"
.done

Subs (inputs)

Address Addr Address Name Type # Description
8 9 left f 1 Desired left motor speed in range -1 to 1
10 11 right f 1 Desired right motor speed in range -1 to 1

Pubs (outputs / config)

Address Name Type # Description
12 limits f 2 Min and max speed limits in range -1 to 1 (default -1 1)
13 port u8 1 Serial port (0..2, default: 2)
14 invert u8 2 Invert axes (0 = normal, 1 = invert)
15 invert u8 1 Switch axes (0=normal, 1=switched i.e. switch left and right motors)
