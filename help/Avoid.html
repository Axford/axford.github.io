Avoid

Base class: Drone

Sail polar calibration module

Usage Guide

Example Configuration

[Avoid=13]

Subs (inputs)

Address Addr Address Name Type # Description
20 21 location f 2 Current location from GPS
32 33 SOG f 1 Speed over ground from GPS
38 39 heading f 1 Current heading from Compass
34 35 course f 1 Course from Nav

Pubs (outputs / config)

Address Name Type # Description
8 adjHeading f 1 Adjusted heading - either passthrough of Heading or generated to avoid obstacle
9 target f 4 Target Lon, Lat, COG and SOG
10 threshold f 2 Thresholds, 0=range beyond which vessels will be ignored, 1=radius within which to treat as a collision
11 vessel u32 4 Number of vessels, 0=total, 1=in range, 2=colliding, 3=mmsi of most urgent potential collision
12 packets u32 2 0=Number of NMEA packets recevied, 1=Number of successfully parsed AIS messages
13 ETC f 1 estimated time to collision
