RFM69Telemetry

Base class: Drone

Manages DroneLink telemetry using an RFM69HW radio module

Example Configuration

[RFM69Telemetry=3]
  name= RFM69
  publish= RSSI,packets,speed,power

Subs (inputs)

Address Addr Address Name Type # Description

Pubs (outputs / config)

Address Name Type # Description
8 RSSI f 1 RSSI of received packets
9 packets u32 3 Packet counters for sent, received and rejected
10 speed f 3 Packet rates per second for sent, received and rejected
11 power f 1 Radio transmit power (-14..20), default 20
12 frequency u32 1 Operating frequency, default 915 Mhz
