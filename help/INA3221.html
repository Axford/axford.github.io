INA3221

Base class: I2CBase

Manages a 3-channel INA3221 I2C power monitoring module

Example Configuration

[INA3221= 4]
  name= "Power"
  bus= 1
  addr= 64
  interval= 1000
  cells= 2
  threshold= 6.4
  status= 1
  publish =current, power, loadV, alarm, shuntV, busV, cellV

Subs (inputs)

Address Addr Address Name Type # Description

Pubs (outputs / config)

Address Name Type # Description
12 current f 3 Current (Amps) for each channel
13 power f 3 Power (Watts) for each channel
14 loadV f 3 Load voltage for each channel
15 cellV f 3 Cell voltage (loadV / cells) for each channel
16 alarm u8 3 Set to 1 when alarm triggered, 0 otherwise - one value per channel
17 cells u8 3 Number of cells (e.g. for a liPo pack) per channel
18 threshold f 3 Threshold voltage below which alarm is triggered (loadV < threshold) per channel
