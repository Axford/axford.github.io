Management

Base class: Drone

Provides overall system management and a pub/sub interface to the DroneModuleManager

Subs (inputs)

Address Addr Address Name Type # Description

Pubs (outputs / config)

Address Name Type # Description
8 hostname c 1 Hostname of the device
9 build c 1 Git commit hash at the time the firmware was built
10 reset u8 1 Set to 1 to trigger a reset
11 heap u32 1 Current size of heap (i.e. free memory)
12 IP u8 4 IP address
13 uptime u32 1 Uptime in seconds
14 publishRate f 1 Rate of messages published per second
15 choked u32 1 Number of times a channel queue has choked (rejected a msg becuse full)
16 discovery u8 1 Enable/disable node discovery process. Set to 1 to enable, or 0 to disable.
17 macro c 1 Set to the path to a macro file (.txt file) to run the associated macro. Note - the macro must be loaded first (e.g. via config.txt).
18 wifi u8 1 Enable/disable wifi. Set to 1 to enable, or 0 to disable.
19 sleep u32 1 How long to sleep in main loop - uses light sleep
20 cpu u32 1 CPU freq in MHz, valid values 240, 160, 80
