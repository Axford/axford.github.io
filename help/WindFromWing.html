WindFromWing

Base class: Drone

Estimate Wind Direction from Wing Compass and Tail position

Usage Guide

Subscribe the Sailor module to the Wind output parameter, instead of a regular wind vane, to use the Wing as an estimator of the Wind angle.

Example Configuration

[ WindFromWing = 12 ]
  name = WindFromWing
  AOA = 15
  $wing = @>Sailor.wing
  $heading = 64>10.22
  publish = wing, heading, AOA, wind

Subs (inputs)

Address Addr Address Name Type # Description
10 11 wing f 1 Wing tail command angle from Sailor.wing
12 13 heading f 1 Current heading of the Wing from the Wing's Compass.heading

Pubs (outputs / config)

Address Name Type # Description
14 AOA f 1 AOA obtained from a Wing value of 1... i.e. multiple Wing command by AOA value to get actual AOA
15 wind f 1 Estimated Wind angle (world coordinates)
